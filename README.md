# MERN Task 

## Project Setup & Installation

### Clone Repository
git clone https://github.com/arpitmalviya26/MERN-TASK-.git
cd MERN-TASK-

---

## Backend Setup
cd Backend
`npm install`

---

## Environment Configuration
Create a `.env` file inside the Backend directory and add the following values:
``
MONGO_URI=your_mongodb_connection_url
PORT=5000
``
- Replace MONGO_URI with your actual MongoDB connection string  
- You may change the port if required  

Run backend:
`npm start`

If configured correctly, the backend will start successfully.

---

## Frontend Setup
`cd Frontend`
`npm install`
`npm start`

The application will run at:
`http://localhost:5173`

---

## Run Order
1️⃣ Start the Backend  
2️⃣ Start the Frontend  
3️⃣ Access the application

---

## Important Notes
- Ensure MongoDB is properly connected   
- Update the API base URL in frontend if required
